# gradient-deployer

A per-host [Restate](https://docs.restate.dev) service that deploys Gradient-built
app binaries by pulling them from the Gradient API and updating a nix profile.

It is a **Restate Virtual Object keyed by slot name**, with one handler,
`Reconcile`. Restate gives us, for free:

- **single-writer per slot** — no two reconciles of the same slot overlap, so
  there is no mutex;
- **durable, auto-retried execution** — each external step is a `restate.Run`
  durable step, so a transient cache miss or a crash mid-deploy resumes from the
  last completed step. No hand-rolled retry loop.

## How it runs

The binary is the Restate SDK server: `gradient-deployer <config.toml>` listens on
`:9080` for invocations from the Restate server. The NixOS module registers the
endpoint with Restate (via `gio.restate.deployments`), so the server discovers the
`Reconcile` handler.

Triggers (all just invoke `Reconcile` on a slot's object via the Restate ingress):

- **Build webhook** — Gradient's `send_web_request` action → webhookcatcher
  (verifies a bearer; Gradient's SSRF guard blocks the loopback ingress directly)
  → local Restate ingress → durable `Reconcile`.
- **Manual** — `gradient-deploy <slot>` (a wrapper generated by the module) POSTs
  to the local Restate ingress.

## What Reconcile does (per slot = object key)

1. `GET /api/v1/projects/<project>/details` → entry-point build IDs.
2. `GET /api/v1/builds/<id>` → require `status == "Succeeded"`, read `output.out`.
3. If that store path differs from the profile's current target: realize it,
   `nix-env --set` the profile, `systemctl restart` the unit. Otherwise no-op.

Steps 1–3 are durable `restate.Run`/`restate.RunVoid` steps. An unknown slot is a
`TerminalError` (no retry); a still-building or empty project is a skip, not a
failure.

## Config (generated by the NixOS module)

```toml
[gradient]
server       = "https://gradient.gio.ninja"
api_key_file = "/run/credentials/gradient-deployer.service/gradient_deployer_api_key"

[restate]
service_name = "gradient-deployer-carbon"   # encodes the host

[slots.yesman]
project      = "default/yesman"   # <org>/<project> for the Gradient API
profile      = "/nix/var/nix/profiles/per-user/deploy/yesman/profile"
restart_unit = "yesman.service"
```
